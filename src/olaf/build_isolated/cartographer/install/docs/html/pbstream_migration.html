Migration tool for pbstream filesÂ¶

With the update of the pbstream serialization format as discussed in RFC-0021, previously serialized pbstream files are not loadable in Cartographer 1.0 anymore.

In order to enable users to reuse previously generated pbstream files, we provide a migration tool which converts pbstreams from Cartographer 0.3 to the new serialization format used in Cartographer 1.0.

The tool is shipped as part of Cartographer’s pbstream tool (source) and once built can be invoked as follows::

cartographer_pbstream migrate old.pbstream new.pbstream

The tool assumes that the first pbstream provided as commandline argument, follows the serialization format of Cartographer 0.3. The resulting 1.0 pbstream will be saved to the second commandline argument location.
